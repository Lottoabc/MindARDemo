Web AR 便签

这是一个单文件的移动端 Web AR Demo。它使用摄像头实时检测蓝色物体，AR 面板本身是可编辑的 2D HTML 卡片，并且会把便签保存到 localStorage。

你现在看到的是“启动说明页”，不是摄像头预览。 要进入 AR，请先用 localhost 或 https 打开本页，然后点击“开始 AR”，允许摄像头权限。

请准备一个颜色明显的蓝色物体，放到镜头前方。识别成功后，摄像头画面会作为背景，便签卡片会浮在蓝色物体上方，你可以直接编辑标题、内容、标签并保存。

怎么进入“HTML 预览状态”：
这不是在编辑器里看静态 HTML，而是用浏览器直接打开本页。最稳的方式是在 VS Code 里用 Live Server，或者在项目目录执行本地静态服务器后访问 http://localhost:端口/index.html。
你会看到什么效果：
对准蓝色物体后，屏幕上会叠加一张半透明、圆角、带阴影的 2D 便签卡片；没有识别到蓝色区域时只会看到扫描提示，不会显示 AR 卡片。

蓝色目标

现在不再识别固定图片了，而是识别画面里的蓝色物体。颜色越纯、背景越简单，识别越稳。你可以先拿一张蓝色纸、蓝色瓶子或蓝色卡片测试。

蓝色物体示意
把一个蓝色物体放到镜头前即可触发 AR。建议蓝色区域占画面有一定面积，背景尽量别太花，这样更容易稳定检测。
